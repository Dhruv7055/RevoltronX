# News Website

## Project Overview

This project is a simple news website that fetches and displays the latest news headlines using the News API. The website is built using HTML, CSS, and JavaScript.

## Files in the Repository

- `index.html`: The main HTML file that structures the webpage.
- `style.css`: The CSS file for styling the webpage.
- `script.js`: The JavaScript file that handles fetching news data from the News API and dynamically updating the webpage.

## Features

- Display the latest news headlines
- Show the article source and publication date
- Error handling for API request failures
